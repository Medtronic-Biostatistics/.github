# Contributor Code of Conduct

This repository adheres to Medtronic code of conduct policies. All contributors are expected to:

-   Treat others with respect and professionalism.
-   Collaborate constructively and transparently.
-   Avoid discriminatory, offensive, or inappropriate behavior.
-   Report any violations promptly to the designated points of contact.

For more details, refer to the official [Medtronic Code of Conduct](https://www.medtronic.com/en-us/our-company/governance/principles-ethics/code-conduct.html).
